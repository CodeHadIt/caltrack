# Development Preferences

## Tech Stack
- Always use the latest version of Next.js with TypeScript
- Use shadcn/ui for component library
- Use Supabase for database and authentication

## UI/Design Guidelines
- Use Google Fonts for typography
- Avoid generic white or black backgrounds
- Create fun, vibrant, and visually engaging interfaces
- Use creative color schemes, gradients, or subtle patterns for backgrounds
- Prioritize personality and delight in the UI

## Project Structure
- Follow Next.js App Router conventions
- Keep components modular and reusable
- Use TypeScript strict mode
